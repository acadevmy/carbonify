# Carbonify

Create and share beautiful images of your source code using [Carbon Now](https://carbon.now.sh/) tool, but with Node.js

It create an headless browser to do the operations using Puppeteer.

## Usage
- `npm install`
- Insert your snippets inside the *input* folder.
- Launch the script with `npm start`
- Images will be saved in the *output* folder.

You can change settings editing config inside the main.js script
